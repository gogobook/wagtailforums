===============
Getting Started
===============

Installation
============

With PyPI
---------

.. code-block:: shell

    pip install wagtailforums==0.1a1


From git
--------

.. code-block:: shell

    pip install -e git://github.com/kaedroho/wagtailforums.git#egg=wagtailforums


Configuration
=============

Add ``wagtailforums`` into your ``INSTALLED_APPS``.

.. code-block:: python

    INSTALLED_APPS = [
        ...

        'wagtailforums',
    ]
